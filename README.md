# [Fragrance Finder](https://noahschipper.net/fragrance)

An interactive fragrance discovery application designed to simplify the process of exploring and comparing perfumes and colognes. Fragrance Finder helps users browse scent profiles, analyze fragrance characteristics, and make more informed decisions when selecting fragrances.

## Features

1. Similar Fragrances: Find fragrances similar to a selected perfume using AI embeddings
2. Search by Accords: Filter fragrances by their main aromatic accords
3. Random Discovery: Get random fragrance recommendations with optional filters
4. Advanced Filtering: Filter by gender, rating, release year, and more
5. User-Focused Interface – Clean, responsive design for easy navigation
